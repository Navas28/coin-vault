# Coin Vault 🛡️

Coin Vault is a premium, privacy-focused personal finance tracker built with React Native and Expo. It allows users to manage their income and expenses entirely on their device, ensuring data ownership and security.

## ✨ Features

- **Local-First Storage**: Powered by SQLite, your financial data never leaves your phone.
- **Smart Dashboard**: Real-time summary of balance, total income, and total expenses.
- **Detailed Analytics**: Beautifully visualized chart to understand spending patterns.
- **Category Management**: Fully customizable categories with a rich icon library.
- **Dark & Light Mode**: Seamless theme switching for a premium look.
- **Transaction History**: Comprehensive list with filtering by date and type.

## 🛠️ Technology Stack

- **Framework**: [Expo](https://expo.dev/) (React Native)
- **Styling**: [NativeWind](https://www.nativewind.dev/) (Tailwind CSS for React Native)
- **Database**: [expo-sqlite](https://docs.expo.dev/versions/latest/sdk/sqlite/)
- **Charts**: [react-native-gifted-charts](https://github.com/Abhinav-K0/react-native-gifted-charts)
- **Navigation**: [Expo Router](https://docs.expo.dev/router/introduction/) (File-based routing)
- **Icons**: [Expo Vector Icons](https://docs.expo.dev/guides/icons/) (Material Design)
- **Fonts**: [Google Poppins](https://fonts.google.com/specimen/Poppins)

## �🚀 Getting Started

1. **Install dependencies**:

   ```bash
   npm install
   ```

2. **Run the project**:

   ```bash
   npx expo start
   ```

3. **Open on your device**:
   Scan the QR code with the Expo Go app (Android) or Camera app (iOS).

## 🔒 Privacy & Security

Coin Vault is designed to be **offline-first**. It does not require an internet connection, does not track your location, and does not upload your data to any cloud service. Your wallet is as secure as your phone.
